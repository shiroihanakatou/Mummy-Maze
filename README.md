# Mummy-Maze
Đồ án môn CSLT cho AI

Hướng dẫn cài đặt:
B1: Clone mã nguồn từ GitHub: https://github.com/shiroihanakatou/Mummy-Maze
B2: Mở terminal tại thư mục chứa project
B3: Cài đặt các thư viện phụ thuộc bằng lệnh pip install -r requirements.txt

Hướng dẫn sử dụng:
Cách 1: Vào build dist chạy exe.
Cách 2: Mở thư mục game chạy main.py
